# Roberta-XGBoost_ForHateSpeech

This project implements a hybrid Transformer-Ensemble approach for detecting hate speech and offensive language on social media. The model leverages the RoBERTa transformer combined with XGBoost to achieve high performance in identifying harmful content.


Dataset Used are:
1. Toxic Tweets Dataset "https://www.kaggle.com/ashwiniyer176/toxic-tweets-dataset"
2. Davidson Dataset "https://doi.org/10.48550/arXiv.1703.04009"
3. Labelled Hate Speech Detection Dataset "https://doi.org/10.6084/m9.figshare.19686954.v1"
